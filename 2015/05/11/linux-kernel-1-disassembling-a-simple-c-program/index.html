汇编基本指令和操作

  • x86中，堆栈的增长方向是从高地址到低地址的。即执行 Push 指令时 ESP 先减小在存入相应的值。
  • Linux下的汇编是AT&T的汇编与INTEL汇编有一些区别，最显而易见的就是MOV指令的方向 AT&T中 mov a b 是 a的值赋给b，而INTEL的相反。
